# nettester_pingmesh
PingMesh as a sample application for NetTester

## Install
Bundle install
```shell
bundle install --binstubs --path=vendor/bundle
```

## Usage
* Run web server.
  * REST API (Web server) is now **DUMMY**.
```shell
bundle exec rackup
```
* Access index.html (default: `http://localhost:9292/`)
